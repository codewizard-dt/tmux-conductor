# Bash Hook Backups

These are the original Bash implementations of the Claude Code lifecycle hooks, retained for reference only after task 011 ported them to JavaScript (`hooks/on-*.js`).

**These scripts are NOT installed by `hooks/install-hooks.sh`.** The installer now registers the `.js` versions under `~/.claude/hooks/tmux-conductor/`.

Kept here so the original shell logic remains accessible via `git log --follow` and direct inspection while the JS ports stabilize.
